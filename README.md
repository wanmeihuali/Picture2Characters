# Picture2Characters
This is a simple python program that transform Image to characters
## Getting Started
### Prerequistes
Python verson > 3.0, PIL
### Usage example
Put any image into the same folder of the program, for example, Lenna.png

Given the input:
```
the picture has been outputed into Lenna.txt, please set the font as Courier New to view the picture
Please input file name of the picture(input exit to exit):
Lenna.png
PNG (512, 512) RGB
Please input height (recommend value: 60 - 80):
30
Please input width (recommend value: 60):
60
Please input file name for output(txt):
Lenna.txt
```
